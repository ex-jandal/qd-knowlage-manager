# QD Knowledge Manager

QD Knowledge Manager is a powerful tool designed for developers, writers, and students to streamline their knowledge management process. It combines a feature-rich Markdown editor with AI-powered summarization to help you capture, organize, and recall information efficiently.

## Features

*   **Advanced Markdown Editor**: A fully-equipped Markdown editor for creating and managing your notes.
*   **Knowledge Management**: Organize your ideas and notes in a structured way.
*   **AI-Powered YouTube Summaries**: Generate concise summaries from YouTube videos to quickly grasp key concepts.
*   **File System Integration**: Browse and manage your local files directly within the application.
*   **Syntax Highlighting**: Code blocks in your notes are beautifully highlighted using highlight.js.

## Technologies Used

*   **Backend**: .NET / C#
*   **Frontend**: Blazor WebAssembly
*   **Styling**: Tailwind CSS
*   **Syntax Highlighting**: highlight.js

## Getting Started

To get the project up and running on your local machine, follow these steps.

### Prerequisites

*   [.NET SDK](https://dotnet.microsoft.com/download)
*   [Node.js and pnpm](https://pnpm.io/installation)

### Installation & Running

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ex-jandal/qd-knowledge-manager.git "QD Knowledge Manager"
    cd "QD Knowledge Manager"
    ```

2.  **Install .NET dependencies:**
    ```bash
    dotnet restore
    ```

3.  **Install frontend dependencies:**
    ```bash
    pnpm install
    ```

4.  **Run the application:**
    ```bash
    dotnet run
    ```

The application will be available at the address specified in the console output (usually `https://localhost:5001`, `http://localhost:5000`, or `http://localhost:5039`).
